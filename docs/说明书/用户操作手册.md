# 多模态心理咨询辅助系统用户操作手册

## 1. 文档信息

- 文档版本：v1.0
- 编写日期：2026-06-14
- 适用对象：普通用户、心理咨询师、课程答辩演示人员

## 2. 系统入口

打开浏览器访问：

```text
http://localhost:5173
```

演示账号：

| 角色 | 邮箱 | 密码 |
| --- | --- | --- |
| 普通用户 | `client@example.com` | `client123` |
| 心理咨询师 | `counselor@example.com` | `counselor123` |

## 3. 页面导航

顶部导航包含：

- `工作台`：登录、上传视频、查看报告、咨询师操作。
- `心理科普`：查看常见心理问题知识和求助提示。
- `项目说明`：查看项目架构、技术路线和非诊断性边界。

## 4. 普通用户操作

### 4.1 登录

1. 打开系统首页。
2. 在登录表单中输入邮箱和密码。
3. 点击登录。
4. 登录成功后进入普通用户工作台。

### 4.2 上传视频

1. 在工作台点击选择文件。
2. 选择一段交流视频。
3. 确认文件大小和文件类型。
4. 点击上传并分析。
5. 等待任务进度更新。

任务处理阶段包括：

- 上传。
- 视频处理。
- 人脸分析。
- 语音分析。
- 融合判断。
- 专家意见。
- 完成。

### 4.3 查看报告

任务完成后，报告会自动显示在页面下方。报告包括：

- 视频时长、抽帧数量、检测人脸、语音检测。
- 综合预判、置信度、风险等级。
- 人脸表情概率和持续时长比例。
- 语音转写、语义情绪、基频、语速和清晰度。
- 专家意见。
- 模型信息和生成时间。

注意：报告仅供心理健康沟通和专业复核参考，不能替代诊断、治疗或危机干预。

### 4.4 查看历史

1. 在普通用户工作台找到分析历史。
2. 点击已完成任务。
3. 页面会加载对应报告。

### 4.5 导出报告

在报告详情区域点击：

- `导出 JSON`：下载结构化报告。
- `导出文本`：下载纯文本报告。

### 4.6 删除任务

1. 在分析历史中找到目标任务。
2. 点击删除按钮。
3. 在确认框中确认删除。
4. 系统会删除任务、报告和本地运行文件。

限制：

- 排队中或分析中的任务不能删除。
- 删除操作不可恢复。

### 4.7 查看授权咨询师

普通用户工作台会展示已授权或已绑定的咨询师列表，便于确认哪些咨询师可以查看自己的历史报告。

## 5. 心理咨询师操作

### 5.1 登录

1. 打开系统首页。
2. 使用咨询师账号登录。
3. 登录成功后进入咨询师工作台。

### 5.2 绑定普通用户

1. 在绑定输入框中输入普通用户邮箱。
2. 点击绑定。
3. 绑定成功后，该用户会出现在关联用户列表中。

说明：

- 只能绑定已存在且角色为普通用户的账号。
- 重复绑定不会创建重复记录。

### 5.3 查看用户历史

1. 在关联用户列表中点击某个用户。
2. 右侧展示该用户的历史分析任务。
3. 点击已完成任务可查看报告详情。

### 5.4 生成咨询师辅助建议

1. 选择已绑定用户。
2. 确认该用户至少有一条完成报告。
3. 点击生成咨询师辅助建议。
4. 页面展示辅助建议和生成时间。

注意：辅助建议仅供专业人员参考，不能替代临床判断。

### 5.5 添加咨询备注

1. 选择已绑定用户。
2. 在备注输入框中输入人工复核信息、沟通重点或下次咨询关注事项。
3. 点击添加备注。
4. 备注会按时间展示在列表中。

### 5.6 查看趋势摘要

选择已绑定用户后，系统会展示该用户已完成报告的风险等级趋势。趋势仅用于辅助观察，不代表正式心理评估结论。

### 5.7 解除绑定

1. 在关联用户列表中点击删除或解除绑定按钮。
2. 确认操作后，该用户从列表中移除。
3. 解除绑定后，咨询师不能再查看该用户历史。

## 6. 心理科普查看

点击顶部 `心理科普`，可以浏览：

- 抑郁相关问题。
- 焦虑相关问题。
- 睡眠问题。
- 压力与适应问题。
- 创伤后应激相关问题。
- 强迫相关问题。
- 需要及时求助的信号。

该页面用于健康教育，不提供自测诊断。

## 7. 项目说明查看

点击顶部 `项目说明`，可以了解：

- 系统目标。
- 架构设计。
- 模型服务。
- 数据流。
- 权限边界。
- 非诊断性声明。

## 8. 常见问题

### 8.1 登录失败怎么办？

确认邮箱和密码是否正确。演示账号为：

- `client@example.com/client123`
- `counselor@example.com/counselor123`

### 8.2 上传后一直处理中怎么办？

视频分析需要等待模型处理。若长时间无变化，可联系维护人员查看后端和模型服务日志。

### 8.3 为什么报告中显示未知或无语音？

可能原因：

- 视频中没有清晰人脸。
- 视频音量过低或没有语音。
- 模型服务未成功分析。
- FFmpeg 未成功抽帧或提取音频。

### 8.4 报告可以作为诊断依据吗？

不可以。系统报告只用于沟通线索和专业复核参考，不能替代心理咨询师、精神科医生或其他专业人员的诊断与治疗。

### 8.5 咨询师为什么看不到某个用户？

咨询师必须先绑定普通用户邮箱。未绑定用户的历史和报告不可访问。

## 9. 退出登录

点击顶部退出按钮即可退出当前账号。退出后本地登录态会清除。
