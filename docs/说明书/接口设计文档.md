# 多模态心理咨询辅助系统接口设计文档

## 1. 文档信息

- 文档版本：v1.0
- 编写日期：2026-06-14
- 接口前缀：`/api`
- 认证方式：Bearer JWT
- 数据格式：JSON，视频上传接口使用 `multipart/form-data`

## 2. 通用约定

### 2.1 请求头

除注册和登录接口外，业务接口均需要：

```http
Authorization: Bearer <access_token>
```

### 2.2 错误响应

FastAPI 默认错误格式：

```json
{
  "detail": "错误说明"
}
```

常见状态码：

| 状态码 | 说明 |
| --- | --- |
| 400 | 请求参数错误 |
| 401 | 未登录或 Token 无效 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 409 | 当前状态不允许操作 |
| 422 | 请求体校验失败 |
| 500 | 服务端异常 |

## 3. 健康检查

### GET `/api/health`

返回服务健康状态。

响应：

```json
{
  "status": "ok"
}
```

## 4. 认证接口

### POST `/api/auth/register`

注册用户。

请求：

```json
{
  "email": "client_a@example.com",
  "password": "client123",
  "role": "client",
  "display_name": "普通用户A"
}
```

字段说明：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `email` | string | 是 | 登录邮箱 |
| `password` | string | 是 | 登录密码 |
| `role` | string | 是 | `client` 或 `counselor` |
| `display_name` | string | 否 | 显示名称 |

### POST `/api/auth/jwt/login`

登录并获取 JWT。

请求类型：`application/x-www-form-urlencoded`

```text
username=client@example.com&password=client123
```

响应：

```json
{
  "access_token": "jwt",
  "token_type": "bearer"
}
```

### GET `/api/users/me`

获取当前用户。

响应字段包含用户 ID、邮箱、角色、显示名称等。

## 5. 普通用户接口

### POST `/api/videos/upload`

上传视频并创建分析任务。仅 `client` 可调用。

请求类型：`multipart/form-data`

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `file` | file | 是 | 视频文件，Content-Type 必须以 `video/` 开头 |

响应：

```json
{
  "task_id": "uuid",
  "status": "queued",
  "stage": "queued",
  "progress": 5,
  "message": "视频已上传，等待后台分析",
  "error": null,
  "created_at": "2026-06-14T10:00:00",
  "updated_at": "2026-06-14T10:00:00",
  "report_url": null,
  "user_id": 1
}
```

### GET `/api/me/tasks`

查询当前普通用户的历史任务。

响应：`TaskOut[]`，按创建时间倒序返回。

### GET `/api/me/counselors`

查询当前普通用户已授权咨询师列表。

### DELETE `/api/me/tasks/{task_id}`

删除当前普通用户自己的已完成或失败任务。

约束：

- 处理中任务返回 409。
- 非本人任务返回 404。
- 删除成功返回 204。

## 6. 任务与报告接口

### GET `/api/tasks/{task_id}`

查询任务状态。普通用户只能查询自己的任务，咨询师只能查询已绑定用户任务。

响应：

```json
{
  "task_id": "uuid",
  "status": "completed",
  "stage": "completed",
  "progress": 100,
  "message": "分析完成",
  "error": null,
  "report_url": "/api/reports/uuid",
  "dominant_emotion": "sad",
  "risk_level": "medium",
  "confidence": 0.67
}
```

### GET `/api/reports/{task_id}`

查询完整分析报告。

核心响应结构：

```json
{
  "task_id": "uuid",
  "video_summary": {},
  "face_emotion": {},
  "speech_features": {},
  "final_prediction": {},
  "expert_advice": "仅供辅助参考...",
  "model_name": "gpt-4.1-mini",
  "prompt_version": "v1",
  "generated_at": "2026-06-14T10:02:00"
}
```

### GET `/api/reports/{task_id}/export?format=json`

导出 JSON 报告。

### GET `/api/reports/{task_id}/export?format=text`

导出纯文本报告。

参数：

| 参数 | 可选值 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `format` | `json`、`text` | `json` | 导出格式 |

## 7. 咨询师接口

### GET `/api/counselor/clients`

查询当前咨询师已绑定普通用户列表。

响应字段：

| 字段 | 说明 |
| --- | --- |
| `id` | 普通用户 ID |
| `email` | 普通用户邮箱 |
| `display_name` | 显示名称 |
| `task_count` | 分析任务数量 |
| `latest_emotion` | 最近一次主情绪 |
| `latest_risk_level` | 最近一次风险等级 |
| `latest_task_at` | 最近一次任务时间 |

### POST `/api/counselor/bindings`

绑定普通用户。

请求：

```json
{
  "client_email": "client@example.com"
}
```

响应：

```json
{
  "client": {
    "id": 1,
    "email": "client@example.com",
    "display_name": "演示普通用户",
    "task_count": 3
  },
  "created": true
}
```

### DELETE `/api/counselor/bindings/{client_id}`

解除绑定。成功返回 204。

### GET `/api/counselor/users/{user_id}/history`

查询已绑定普通用户历史任务。

### POST `/api/counselor/users/{user_id}/assistance-draft`

生成咨询师辅助建议草稿。

响应：

```json
{
  "user_id": 1,
  "assistance": "仅供专业人员参考...",
  "generated_at": "2026-06-14T10:00:00"
}
```

### GET `/api/counselor/users/{user_id}/notes`

查询咨询师对指定用户的备注列表。

### POST `/api/counselor/users/{user_id}/notes`

新增咨询备注。

请求：

```json
{
  "content": "下次咨询优先核实睡眠和近期压力源。"
}
```

### GET `/api/counselor/users/{user_id}/trend`

查询用户风险趋势。

响应：

```json
{
  "user_id": 1,
  "points": [
    {
      "task_id": "uuid",
      "created_at": "2026-06-14T10:00:00",
      "emotion": "sad",
      "risk_level": "medium",
      "confidence": 0.67
    }
  ]
}
```

## 8. 内部模型服务接口

### DeepFace `POST /analyze`

后端内部调用，输入任务 ID、抽帧目录和帧数量，输出表情分析结果。

### SenseVoice `POST /analyze`

后端内部调用，输入任务 ID、音频路径和时长，输出转写、语义情绪和声学特征。

## 9. 前端轮询策略

上传成功后，前端每 1 秒调用一次 `GET /api/tasks/{task_id}`，最多轮询 180 次。任务完成后立即调用 `GET /api/reports/{task_id}` 拉取报告；任务失败时展示 `error` 或 `message`。
