# 多模态心理咨询辅助系统运维手册

## 1. 文档信息

- 文档版本：v1.0
- 编写日期：2026-06-14
- 适用对象：开发人员、演示维护人员、课程答辩现场保障人员

## 2. 日常启动与停止

### 2.1 启动系统

```powershell
docker compose up -d --build
```

### 2.2 停止系统

```powershell
docker compose down
```

### 2.3 查看状态

```powershell
docker compose ps
```

### 2.4 查看日志

```powershell
docker compose logs -f backend
docker compose logs -f frontend
docker compose logs -f deepface
docker compose logs -f sensevoice
docker compose logs -f postgres
```

## 6. 备份与恢复

### 6.1 数据库备份

```powershell
docker compose exec postgres pg_dump -U emotion -d emotion > emotion_backup.sql
```

### 6.2 数据库恢复

```powershell
Get-Content .\emotion_backup.sql | docker compose exec -T postgres psql -U emotion -d emotion
```

```powershell

```

## 7. 日志与排查

### 7.1 上传后任务失败

排查步骤：

1. 查看后端日志：`docker compose logs -f backend`。
2. 检查视频是否为浏览器和 FFmpeg 可识别格式。
3. 检查 `storage/uploads` 是否有对应视频文件。
4. 检查 `storage/frames/{task_id}` 是否生成图片。
5. 检查 `storage/audio/{task_id}.wav` 是否生成音频。

### 7.2 表情分析失败

排查步骤：

1. 查看 DeepFace 日志。
2. 确认首次启动是否正在下载模型权重。
3. 检查抽帧图片是否存在。
4. 检查视频中是否有人脸或画面是否过暗。

### 7.3 语音分析失败

排查步骤：

1. 查看 SenseVoice 日志。
2. 确认首次启动是否正在下载 ModelScope 模型。
3. 检查音频文件是否存在。
4. 检查视频是否包含有效语音。

### 7.4 LLM 无输出或输出兜底建议

排查步骤：

1. 检查 `.env` 中 `OPENAI_API_KEY` 是否配置。
2. 检查 `OPENAI_BASE_URL` 是否可访问。
3. 检查 `OPENAI_MODEL` 是否为兼容接口支持的模型。
4. 查看后端日志中 LLM 调用错误。

### 7.5 前端无法访问后端

排查步骤：

1. 确认 `VITE_API_BASE_URL` 为 `http://localhost:8000`。
2. 确认后端端口 8000 未被占用。
3. 检查 `BACKEND_CORS_ORIGINS` 是否包含前端地址。
4. 在浏览器开发者工具中查看网络错误。

## 8. 安全运维要求

- 生产环境必须修改 `AUTH_SECRET_KEY`。
- 不要将 `.env`、API Key、真实用户视频和报告提交到 Git。
- 不在日志中输出完整 Token、API Key 或敏感咨询文本。
- 真实使用场景应启用 HTTPS、访问审计、数据加密和数据删除策略。
- 咨询师绑定流程在生产环境应增加普通用户授权确认。

## 9. 容量与清理

### 9.1 文件增长来源

- 上传视频可能占用大量磁盘。
- 抽帧图片按视频时长增长。
- 音频文件与报告文件随任务增长。
- 模型缓存首次下载后会占用较大空间。

### 9.2 清理建议

课程演示环境：

- 演示前保留样例数据。
- 演示后可手动删除不需要的任务或清空 `storage`。

生产环境：

- 增加任务文件保留周期。
- 增加对象存储生命周期策略。
- 增加用户主动删除和管理员审计删除能力。

## 10. 应急处理

| 问题 | 快速处理 |
| --- | --- |
| 服务无响应 | `docker compose restart` |
| 数据库未 healthy | 查看 postgres 日志，确认 volume 和端口 |
| 模型下载过慢 | 提前启动并缓存模型，或配置国内镜像 |
| 任务卡在处理中 | 查看后端日志，必要时重启后端并重新上传 |
| 演示现场无法调用 LLM | 使用系统兜底建议完成流程 |
