# 多模态心理咨询辅助系统测试用例与验收方案

## 1. 文档信息

- 文档版本：v1.0
- 编写日期：2026-06-14
- 适用阶段：本地测试、课程答辩前验收、交付验收

## 2. 测试范围

测试范围包括：

- 前端登录注册、导航、上传、历史、报告、导出、咨询师工作台。
- 后端认证、任务、报告、咨询师和导出接口。
- 数据库迁移、数据持久化和权限隔离。
- DeepFace、SenseVoice 和 LLM 兜底链路。
- Docker Compose 部署启动。

不包含：

- 医学诊断准确性测试。
- 高并发压测。
- 第三方模型临床有效性验证。

## 3. 测试环境

| 项目 | 要求 |
| --- | --- |
| 操作系统 | Windows 10/11 或支持 Docker 的 Linux/macOS |
| Docker | 已安装 Docker Desktop 或 Docker Engine |

## 5. 功能测试用例

### 5.1 注册登录

| 编号 | 用例 | 前置条件 | 步骤 | 预期结果 |
| --- | --- | --- | --- | --- |
| TC-001 | 普通用户登录 | 服务已启动 | 输入 `client@example.com/client123` 登录 | 登录成功，进入普通用户工作台 |
| TC-002 | 咨询师登录 | 服务已启动 | 输入 `counselor@example.com/counselor123` 登录 | 登录成功，进入咨询师工作台 |
| TC-003 | 错误密码登录 | 服务已启动 | 输入错误密码 | 页面展示登录失败提示 |
| TC-004 | 新用户注册 | 邮箱未注册 | 填写邮箱、密码、角色、显示名并提交 | 注册后自动登录 |

### 5.2 普通用户视频分析

| 编号 | 用例 | 前置条件 | 步骤 | 预期结果 |
| --- | --- | --- | --- | --- |
| TC-005 | 上传视频 | 普通用户已登录 | 选择视频文件并点击上传 | 返回任务 ID，进度进入排队或分析中 |
| TC-006 | 非视频文件上传 | 普通用户已登录 | 选择非视频文件上传 | 后端返回 400，页面展示错误 |
| TC-007 | 任务轮询 | 已上传视频 | 等待任务处理 | 进度阶段依次更新，最终完成或失败 |
| TC-008 | 查看报告 | 任务已完成 | 打开报告 | 展示视频摘要、表情、语音、预判和专家意见 |
| TC-009 | 导出 JSON | 报告已打开 | 点击导出 JSON | 浏览器下载 JSON 文件 |
| TC-010 | 导出文本 | 报告已打开 | 点击导出文本 | 浏览器下载 TXT 文件 |
| TC-011 | 删除历史任务 | 任务已完成或失败 | 点击删除并确认 | 任务从历史列表消失，文件和数据库记录被清理 |
| TC-012 | 删除处理中任务 | 任务处理中 | 点击删除 | 按钮禁用或后端返回 409 |

### 5.3 咨询师工作台

| 编号 | 用例 | 前置条件 | 步骤 | 预期结果 |
| --- | --- | --- | --- | --- |
| TC-013 | 绑定普通用户 | 咨询师已登录 | 输入 `client@example.com` 绑定 | 关联用户列表出现该用户 |

| TC-017 | 生成辅助建议 | 用户有历史报告 | 点击生成咨询师辅助建议 | 展示辅助建议和生成时间 |

### 5.4 权限测试

| 编号 | 用例 | 前置条件 | 步骤 | 预期结果 |
| --- | --- | --- | --- | --- |
| TC-021 | 未登录访问报告 | 无 Token | 调用报告接口 | 返回 401 |

### 5.5 部署与运行测试

| 编号 | 用例 | 前置条件 | 步骤 | 预期结果 |

## 6. 验收标准

系统满足以下条件即可通过课程设计验收：

- Docker Compose 可以启动前端、后端、DeepFace、SenseVoice 和 PostgreSQL。
- 演示账号可以登录。
- 普通用户可以上传视频并看到任务状态变化。
- 系统可以生成或兜底生成完整报告。
- 报告包含视频摘要、表情、语音、综合预判、专家意见和非诊断性声明。
- 普通用户可以查看历史、导出报告并删除已完成或失败任务。
- 咨询师可以绑定普通用户、查看历史、生成辅助建议、添加备注并查看趋势。
- 权限隔离有效，不能越权访问未授权数据。
- 文档、截图、部署和运维资料齐全。

## 7. 缺陷分级

| 级别 | 说明 | 示例 |
| --- | --- | --- |
| P0 | 阻断验收 | 服务无法启动，登录完全不可用 |
| P1 | 严重缺陷 | 上传后必然失败，权限隔离失效 |
| P2 | 一般缺陷 | 单个导出格式失败，个别页面提示不准确 |
| P3 | 轻微缺陷 | 样式错位、文案瑕疵、不影响主流程 |
